# pokeDMGcal
pokemon_damage_calculator

2020.11.10
概要
・ステータス計算
・ダメージ計算
・耐久調整

オプションとして
素早さ表

一番作りたいのは耐久調整
攻撃A+B+C+Dを耐える
または攻撃A or B or C or Dを耐える
最小の努力値∼HPや防御が最大といった具合

最大ダメージの物理攻撃を計算 -> 耐久計算 -> 耐えるパターンから特殊攻撃の耐えるパターンを求める
といった具合
攻撃回数も設定、インファ1耐えかつ10万2耐えなども計算可能に
乱数の容認範囲、100%~85%まで, 1%区切り

サーバー aws
表面 HTML, CSS, Javascript, React
